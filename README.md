# All-in-one Download — Rotten Tomatoes

Add-on for the [**All-in-one Download**](https://github.com/tcacamou-ops/all-in-one-download) WordPress plugin that adds support for **Rotten Tomatoes** URLs.

Given a link to a movie or TV show page on Rotten Tomatoes, this module automatically retrieves:

- The **title** of the movie or TV show
- The **poster** (cover image)
- The **media type** (movie or TV show)

Simply paste a Rotten Tomatoes URL into All-in-one Download and the media information will be filled in automatically.
